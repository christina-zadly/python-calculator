# Python Calculator

A simple calculator built with Python as part of my Python learning journey.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Input validation
* Division by zero handling
* Function-based design

## Technologies

* Python 3

## How to Run

```bash
python calculator.py
```
